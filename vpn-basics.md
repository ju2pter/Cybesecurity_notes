# VPN Basics

## What is a VPN?
A Virtual Private Network (VPN) creates a secure encrypted connection between a device and a network over the internet.

## Why companies use VPNs
- Secure remote access
- Protect sensitive data
- Prevent unauthorized access
- Enable safe remote work

## Common types
### Remote Access VPN
Used by employees to connect securely to company networks.

### Site-to-Site VPN
Connects different company locations securely.

## Security benefits
- Encryption of data
- Protection against interception
- Secure communication channels.
